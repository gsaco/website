DML_NN

Double/Debiased Machine Learning with neural networks. Compares activation functions (ReLU, Sigmoid, Tanh) across 1-3 layer architectures. Implements cross-fitting (K=2 folds) for valid causal inference on partially linear models. Evaluates 48 model combinations including OLS, LASSO, Random Forest, and NNs.

Double ML Neural Networks Cross-fitting Causal Inference

Lasso_Potential_Outcomes_RCTs

LASSO-assisted covariate adjustment for RCTs using the Potential Outcomes Framework. Compares low-dimensional (5 features) vs. high-dimensional (400+ engineered features) specifications. Implements cross-validation for regularization, ATE estimation, and randomization inference.

LASSO Potential Outcomes RCT ATE

Classification Trees (CART), Causal Forests, and Random Forests for heterogeneous treatment effects. Implements 4-fold CV with complexity tuning, tree pruning (Œ± = 0.0001643517), and CATE estimation with tercile stratification for patient response groups.
